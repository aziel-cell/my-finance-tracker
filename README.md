# 💰 Personal Finance Tracker

Aplikasi pelacak keuangan pribadi berbasis web yang dibangun dengan **Python** dan **Streamlit**. Aplikasi ini membantu Anda mencatat pengeluaran harian, mengkategorikannya, dan melihat visualisasi data pengeluaran secara real-time.

---

## ✨ Fitur Utama
- **Input Transaksi:** Catat pengeluaran berdasarkan tanggal, kategori, dan deskripsi.
- **Dashboard Ringkasan:** Lihat total pengeluaran dan jumlah transaksi secara instan.
- **Visualisasi Interaktif:** Grafik Pie Chart otomatis untuk menganalisis kategori pengeluaran terbesar menggunakan Plotly.
- **Penyimpanan Data Lokal:** Data tersimpan secara permanen dalam format `.csv`, sehingga tidak hilang saat aplikasi ditutup.

## 🛠️ Tech Stack
- **Bahasa Pemrograman:** [Python](https://www.python.org/)
- **Framework Web:** [Streamlit](https://streamlit.io/)
- **Olah Data:** [Pandas](https://pandas.pydata.org/)
- **Visualisasi:** [Plotly Express](https://plotly.com/python/plotly-express/)
